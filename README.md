# pure-php-stp

CRUD with prue PHP

## Features

* Editor

## How-to

* Intstall Xampp
* open Apache and MySQL
* add project folder in <code>c:/xampp/htdocs/</code>
* go to <code>http://localhost/phpmyadmin/</code>
* create the **assignment** database

* import **database.sql** 

* run <code>http://localhost/folder-name</code>
